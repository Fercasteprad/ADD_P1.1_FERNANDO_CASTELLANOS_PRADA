
Método desbloquear:

Previamente tengo creado un arrayList de usuarios que guardan, un Id con la línea correspondiente al del fichero de acceso, un booleano para saber si están bloqueados o no
y un booleano para saber si son administradores.

Realizo un while leyendo cada línea del documento acceso y a la vez voy recorriendo el arraylist mirando si están bloqueados, en el caso de estar bloqueado lo muestro
por pantalla y me guardo en un arrayList de enteros la posición del primer arraylist.

Una vez elije el administrador a que usuario desbloquear, utilizo el arrayList de enteros para poner a true el estado y ya estaría desbloqueado.
